VERTTYPE


#define VERTTYPE   GLfloat
 

*************************************************************************** OpenGL ES function macros to abstract the profile used (Common or Common-Lite) Defines a set of shims of the form myglXXX which can allow on-the-fly conversion from a floating point format to fixed-point OpenGL ES entrypoints. If the flag PVRT_FIXED_POINT_ENABLE is defined, inputs to the function are converted to fixed point and the the OpenGL ES Lite version of the function is then called If the flag PVRT_FIXED_POINT_ENABLE is not defined, the macros directly default to their Common profile counterpart.
